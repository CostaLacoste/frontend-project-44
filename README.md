# Игры разума

Игры разума это набор 5 консольных игр, которые включают в себя:
* Калькулятор. Арифметические выражения, которые необходимо вычислить.
* Прогрессия. Поиск пропущенных чисел в последовательности чисел.
* Определение чётного числа.
* Определение НОД.
* Определение простого числа.


## Игра калькулятор

В этой игре нужно найти сумму/разность/произведение/частное двух чисел. Игра завершиться после 3 правильных ответом, либо одного неправильного.
[![asciicast](https://asciinema.org/a/ZYS3XQBE3CPa7AQMruhH32Xaw.svg)](https://asciinema.org/a/ZYS3XQBE3CPa7AQMruhH32Xaw)

## Игра прогрессия

В этой игре нужно найти недостающее число арифметической прогрессии. Игра завершится после 3 правильных ответов.
[![asciicast](https://asciinema.org/a/opcx1Y4MkekxtzsKbgLZ7Vigv.svg)](https://asciinema.org/a/opcx1Y4MkekxtzsKbgLZ7Vigv)

## Игра определение чётного числа
В этой игре нужно определить является ли число чётным. Игра завершается после 3 правильных ответов.
[![asciicast](https://asciinema.org/a/K20BjppBaq9BP6tPRdo0O9Hkw.svg)](https://asciinema.org/a/K20BjppBaq9BP6tPRdo0O9Hkw)

## Игра определение НОД
В этой игре нужно определить наибольший общий делитель 2 слагаемых. Игра завершается после 3 правильных ответов.
[![asciicast](https://asciinema.org/a/U3DkeIHAGPdb7bKKOPQrwIy8O.svg)](https://asciinema.org/a/U3DkeIHAGPdb7bKKOPQrwIy8O)

## Игра определение простого числа
В этой игре нужно ответить yes если число простое или no если число имеет больше 2 делителей.
[![asciicast](https://asciinema.org/a/d0JGFFCI0ljt69yMSXI8sOAEL.svg)](https://asciinema.org/a/d0JGFFCI0ljt69yMSXI8sOAEL)
